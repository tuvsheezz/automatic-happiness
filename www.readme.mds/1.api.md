#### Setting the API & Api versioning
```sh
  mkdir app/controllers/api
  mkdir app/controllers/api/v1
```

 Add namespaces to config/routes.rb

```rb
namespace :api, defaults: { format: :json } do
  namespace :v1 do
    #v1 routes goes here...
  end
end
```
